1. Project Introduction

The Weather Analytics Portal bridges the gap between raw meteorological data and actionable insights.

  • Provides automated clean data intake, purging anomalies.
  • Facilitates robust search & analysis through interactive widgets.
  • Generates insights automatically, reducing manual weather interpretation.

2. Architecture & Tech Stack

Designed with modularity and separation of concerns:

Backend

Django (Python)
Pandas & NumPy
SQLite

Frontend

HTML5 & Custom CSS
Bootstrap 5
Chart.js (CDN)

Exports

ReportLab PDF Engine
Pandas CSV exporter

3. Pandas & NumPy Processing

How backend calculations operate on data streams:

  • Data Cleaning: Handles missing/null temperatures, formats dates seamlessly.
  • Extreme Weather Detection: Identifies heatwaves (≥35°C), storms, and flood hazards.
  • Correlation Metrics: Calculates Pearson correlation coeff between temperature and humidity.

4. Visualizations & Analytics

Providing real-time telemetry representation:

  • Metrics: Dynamic cards showing total records, average, max, and min temp.
  • Interactive Charts: Line trends for temperature coupled with precipitation histograms.
  • Scatter Mapping: Direct visualization of humidity/temperature interactions.
